
# CodeAlpha - C Programming Projects

This repository contains a collection of C language projects built as part of my coding practice. Each program demonstrates different concepts of C like file handling, functions, loops, and data structures.

## 📂 Projects Included

- **Bank Management System** `bank.c` 
    - Create new accounts, deposit, withdraw, check balance
    - Data stored in `bank.dat` file

- **Simple Calculator** `calculator.c`
    - Performs addition, subtraction, multiplication, division
    - Command line based interface

- **Matrix Operations** `matrix.c`
    - Matrix addition, subtraction, multiplication
    - User input for matrix size and elements

- **Student Record System** `student.c`
    - Add, view, search, delete student records
    - Data stored in `students.dat` file

## 🚀 How to Compile & Run

1. Clone the repo: `git clone https://github.com/chennamsettibindu-star/codeAlpha.git`
2. Compile any file: `gcc filename.c -o filename.exe`
3. Run: `./filename.exe` or `filename.exe` on Windows

Example:
gcc bank.c -o http://bank.exe
./bank.exe
